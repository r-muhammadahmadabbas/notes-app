## ğŸ“– Overview

NotesApp is a lightweight productivity tool designed for capturing quick
thoughts, ideas, and reminders.\
It focuses on simplicity, performance, and clean UI/UX using modern web
technologies.

The backend is built with **FastAPI**, offering high-performance
asynchronous APIs, while **MongoDB** provides flexible NoSQL storage.\
The frontend is rendered server-side using **Jinja2** and styled with
**Bootstrap 5**, including glassmorphism and masonry layouts.
